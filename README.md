# Emdify

Emdify is a simple JavaScript library for hasing the user input text.

## Installation

No dependencies (except node v10+).

```bash
npm i emdify
```

## Usage

```javascript
import { md5, MD5 } from "emdify";

console.log(md5("Hello!")); // 952d2c56d0485958336747bcdd98590d

console.log(MD5("Hello!")); // 952D2C56D0485958336747BCDD98590D
```
